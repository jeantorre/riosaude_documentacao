Schema e Dicionário de Dados

Schema da tabela irs_fat_alta

Bases: DataFrameModel

Esquema de validação de dados utilizando Pandera para as informações extraídas do banco de dados referentes à alta de pacientes.

Attributes:

Name Type Description
alta_gid Series[str]

Chave única do registro de alta hospitalar.
boletim_gid Series[str]

Chave única do Boletim de Atendimento Médico (BAM).
estabelecimento_gid Series[str]

Chave única do estabelecimento de saúde.
mal_id Series[int]

Chave única dos motivos de alta.
motivo_saida Series[str]

Motivo pelo qual o paciente recebeu alta.
alta_administrativa Series[Timestamp]

Data/hora que paciente recebe alta através do sistema TIMed.
status Series[str]

Indica se a alta foi efetuada ou cancelada.

Schema da tabela irs_fat_atendimento

Bases: DataFrameModel

Esquema de validação de dados utilizando Pandera para as informações extraídas do banco de dados referentes ao atendimento do paciente.

Attributes:

Name Type Description
atendimento_gid Series[str]

Chave única para identificação de cada atendimento.
matd_id Series[str]

Chave única dos tipos de atendimento.
boletim_gid Series[str]

Chave única do Boletim de Atendimento Médico (BAM).
estabelecimento_gid Series[str]

Chave única do estabelecimento de saúde.
cid_id Series[int]

Chave única dos diferentes tipos de CID.
esp_id Series[int]

Chave única das especialidades.
data_inicio Series[Timestamp]

Data/hora de início do atendimento.
data_fim Series[Timestamp]

Data/hora do fim do atendimento.
fat_paciente_rede_id Series[int]

Chave única que identifica cada paciente no banco.
prf_id Series[int]

Chave única que identifica cada profissional no banco.

Schema da tabela irs_fat_boletim

Bases: DataFrameModel

Attributes:

Name Type Description
boletim_gid Series[str]

Chave única do Boletim de Atendimento Médico (BAM).
tpa_id Series[int]

Chave única do tipo de atendimento médico.
estabelecimento_gid Series[str]

Chave única do estabelecimento de saúde.
data_entrada Series[Timestamp]

Data/hora de entrada do paciente na unidade de saúde.
data_internacao Series[Timestamp]

Data/hora do início da internação.
tpu_id Series[int]

Chave única do tipo de Unidade.
tpe_id Series[int]

Chave única do tipo de Registro.
interno Series[int]

Identificador se o paciente é interno ou externo.
esp_id Series[int]

Chave única do tipo de especialidade médica.
